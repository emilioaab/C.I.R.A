# C.I.R.A - Cloud Incident Response Analyzer

## Project Description
Cloud security monitoring and incident response system for Multi-Cloud environments (AWS, Azure, GCP).
Bridges the gap between CSPM (static monitoring) and IR (dynamic response).

## Current Status
**In Development** - MVP Phase

### Completed:
- Python 3.12 environment setup
- AWS API connection
- Project structure
- Dependencies installation

### In Progress:
- Database setup
- Basic dashboard
- Anomaly detection engine

## Requirements
- Python 3.12+
- AWS Account with appropriate IAM permissions
- Ubuntu/Linux environment
